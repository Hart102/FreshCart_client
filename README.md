# FreshCart Project

## Features:

1. User Registration
2. User Login
3. Edit User Profile
4. Password Reset
5. Add to cart/Remove from cart
6. Payment for selected product

7. Create Product
8. Edit Product
9. Delete Product
10. Create Category
11. Edit Category/Delete category
12. View user Order
